# Liang (Eric) Yang - Personal Website

A clean, minimal personal website showcasing research and publications.

## About

This is the personal website of Liang (Eric) Yang, Senior Research Manager and Principal Researcher at Apple.

## Deployment to GitHub Pages

### Option 1: Deploy to your existing ericlyang.github.io repository

1. Copy all files from this directory to your `ericlyang.github.io` repository
2. Commit and push:
```bash
git add .
git commit -m "Update personal website"
git push origin main
```

### Option 2: Create a new repository

1. Create a new repository on GitHub named `ericlyang.github.io`
2. In this directory, run:
```bash
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/EricLYang/ericlyang.github.io.git
git push -u origin main
```

3. Go to your repository settings on GitHub
4. Navigate to Pages section
5. Set source to "Deploy from a branch"
6. Select "main" branch and "/ (root)" folder
7. Click Save

Your website will be available at: https://ericlyang.github.io/

## Local Preview

Simply open `index.html` in your web browser to preview the website locally.

## Updating Content

- Edit `index.html` to update your bio, publications, and links
- Modify `style.css` to change the appearance
